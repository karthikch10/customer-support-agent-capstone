# Multi-Agent Customer Support Assistant (Capstone - MVS)

**Author:** Karthik Ch

## Short description
A multi-agent customer support system that classifies incoming messages, retrieves relevant knowledge, attempts automated resolution, and creates structured tickets for escalation. Demonstrates: multi-agent orchestration, tools (ticket creation + search), sessions & memory, observability (logs/traces), and LLM-based evaluation.

## Quick run (Kaggle Notebook)
1. Open `notebooks/demo_notebook.ipynb` (public Kaggle notebook).
2. Run cells from top to bottom.
3. Example demo conversations are included in the notebook.
